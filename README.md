# Very Simple Gray Theme for Chrome

This project contains the editable source files for a local unpacked Chrome theme.

## Load Unpacked

1. Open Chrome and go to `chrome://extensions`.
2. Turn on **Developer mode**.
3. Click **Load unpacked**.
4. Select `/Users/pepe/development/chrome-gray-theme`.

If you change `manifest.json` or any image, reopen `chrome://extensions` and click **Reload** on the theme.
